
# Game Development Challenges

This section collects links to coding challenges, exercises, and competitions that help improve your game development skills. These challenges are designed to be practical, fun, and educational, similar to events like Advent of Code, game jams, and other programming contests.

## Examples of Challenges

- **Advent of Code**: Annual programming challenge with daily problems in December. [Advent of Code](https://adventofcode.com/)
- **20 Games Challenge**: Complete 20 small games to improve your skills. [20 Games Challenge](https://20_games_challenge.gitlab.io/)
- **Game Jams**: Short-term events where developers create games from scratch. Popular jams include Ludum Dare, Global Game Jam, and GMTK Game Jam.
- **Coding Exercises**: Websites like LeetCode, HackerRank, and Codewars offer problems that can be adapted for game development practice.
- **Open Source Game Challenges**: Contribute to open source game projects or participate in community-driven challenges.

Feel free to add more links to challenges, competitions, or exercises that help you learn and practice game development skills!

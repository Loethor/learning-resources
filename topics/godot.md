# Resources for Godot Game Engine

## Official Documentation
- [Godot Engine Documentation](https://docs.godotengine.org/en/stable/) - The official documentation for Godot, covering everything from getting started to advanced topics.

## Tutorials
[Godot Tutorials by GDQuest](https://gdquest.com/) - A collection of high-quality tutorials and courses for learning Godot.

## Community Resources
- [Godot Community Forums](https://godotforums.org/) - A place to ask questions and share knowledge with other Godot users.
- [Godot Discord Server](https://discord.gg/godotengine) - Join the Godot community on Discord for real-time discussions and support.

## Sample Projects
- [Godot Sample Projects](https://github.com/godotengine/godot-demo-projects) - A collection of demo projects showcasing various features of Godot.

## Shaders
- [Godot Shaders](https://godotshaders.com/) - A community-driven collection of shaders for the Godot engine.

## Plugins and Tools
- [Godot Asset Library](https://godotengine.org/asset-library/asset) - A repository of community-created assets and plugins for Godot.

## Using This Vault

### Navigation
- Click on any `[[link]]` to navigate between notes
- Use backlinks panel (right sidebar) to see what references the current note
- Press `Ctrl+O` (or `Cmd+O` on Mac) for quick file switching
- Press `Ctrl+P` (or `Cmd+P`) for command palette

### Tagging System
Use these tags to filter and organize content:
- `#tutorial` - Step-by-step guides and how-tos
- `#gamedev` - Game development related content
- `#godot` - Godot engine specific resources
- `#challenge` - Practice problems and exercises
- `#blog` - Blog posts and articles
- `#community` - Communities and forums
- `#productivity` - Productivity and workflow tips
- `#planning` - Project planning and organization

### Graph View
Press `Ctrl+G` (or `Cmd+G`) to see the knowledge graph showing connections between notes.

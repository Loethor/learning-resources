# Learning Resources

Welcome to my Learning Resources repository! This project is designed to accumulate and organize valuable links to articles, videos, and other resources that helped me learn specificy topics. There are topics such as game development, software architecture, and optimization.

## Purpose

The goal of this repository is to provide a centralized collection of resources covering various topics in game development. Whether you're a beginner or an experienced developer, you'll find useful materials to enhance your knowledge and skills.

## Topics Covered

- **Blogs**: Read insightful blogs and personal websites that provide tutorials and perspectives on game development. [View Blog Resources](topics/blogs.md)
- **Godot**: Discover resources specific to the Godot game engine, including guides, articles, and community contributions. [View Godot Resources](topics/godot.md)
- **Challenges**: Find coding challenges, game jams, and competitions to practice and improve your game development skills. [View Challenges](topics/challenges.md)
